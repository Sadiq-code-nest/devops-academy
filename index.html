01

Developer pushes to GitHub

A feature branch is merged into main. GitHub fires a webhook to Jenkins immediately.

02

Jenkins runs the CI pipeline

Declarative Jenkinsfile runs: lint → unit tests → code coverage. Pipeline fails fast if anything breaks.

03

Docker image built and pushed to ECR

Multi-stage Dockerfile produces a lean production image tagged with the Git SHA, then pushed to AWS ECR.

04

Ansible deploys to EC2 (zero downtime)

Jenkins triggers an Ansible playbook that pulls the new image, starts the container, and only cuts traffic once the health check passes.

05

Nginx routes traffic, CloudWatch monitors

Nginx terminates SSL and reverse-proxies to the container. CloudWatch alarms fire if CPU or error rate spikes.
